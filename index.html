📲 Comment payer avec Orange Money

✅ Le plus simple : chez un kiosque Orange Money

  1. Cherchez un vendeur/kiosque avec le logo Orange Money (il y en a partout à Bamako, souvent à côté des boutiques).
  2. Donnez-lui ce numéro : 94134408 et ce montant : 0 FCFA.
  3. Remettez-lui l'argent en espèces, il fait le transfert pour vous (petits frais possibles, quelques dizaines de FCFA).
  4. Gardez le message ou le reçu qu'il vous montre, c'est votre preuve de paiement.

📱 Si vous avez déjà un compte Orange Money sur votre téléphone

  1. Composez #144# sur votre téléphone, puis suivez le menu Transfert d'argent.
  2. Entrez le numéro 94134408, puis le montant 0 FCFA.
  3. Entrez votre code secret Orange Money, puis validez.

📲 Comment payer avec Wave

✅ Le plus simple : chez un kiosque Wave

  1. Cherchez un vendeur/kiosque avec le logo bleu Wave (très courant à Bamako).
  2. Donnez-lui ce numéro : 94134408 et ce montant : 0 FCFA.
  3. Remettez-lui l'argent en espèces, il fait le transfert pour vous.
  4. Gardez le reçu ou le SMS de confirmation qu'il vous montre.

📱 Si vous avez déjà l'application Wave sur votre téléphone

  1. Ouvrez l'application Wave, appuyez sur Envoyer de l'argent.
  2. Entrez le numéro 94134408 et le montant 0 FCFA, puis confirmez.
